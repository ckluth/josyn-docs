Deployment & Operations

Describes how JOSYN gets onto a machine and stays running. This chapter answers: how is the system installed, how is it bootstrapped, how is it upgraded, and what must an operator know to run it day-to-day.

In Scope

  • Bootstrapping sequence
  • Upgrade and rollback procedures
  • Operator conventions and responsibilities
  • Database backup and migration

Out of Scope

  • Certificate and identity setup → see 04-identity-and-security/
  • Compliance and audit logging → see 06-company-fit/
